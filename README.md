# Computer Vision Lab

A collection of computer vision and image processing projects implemented in MATLAB.

This repository includes multiple practical experiments related to:

* Motion detection
* Frame differencing
* Gaussian Mixture Models (GMM)
* Object tracking
* HOG feature extraction
* CNN image classification
* Robot treasure hunting
* Path planning
* Visual tracking

---

# Project Overview

The purpose of this project is to explore fundamental and advanced computer vision techniques through practical implementations and experiments.

The repository contains multiple laboratory exercises and mini-projects involving:

* Video processing
* Image segmentation
* Motion analysis
* Feature extraction
* Object recognition
* Tracking systems

---

# Main Experiments

## Frame Differencing Motion Detection

Implemented frame differencing methods for detecting moving objects from video sequences.

Features:

* Background subtraction
* Motion segmentation
* Threshold analysis

---

## Gaussian Mixture Model (GMM)

Implemented GMM-based foreground extraction and background modelling.

Features:

* Adaptive background modelling
* Noise reduction
* Dynamic foreground detection

---

## HOG Feature Extraction

Implemented Histogram of Oriented Gradients (HOG) for image feature extraction and recognition.

Applications:

* Shape detection
* Object recognition

---

## CNN Classification

Implemented convolutional neural network (CNN) based image classification experiments.

Features:

* Training and testing datasets
* Confusion matrix evaluation
* Classification accuracy analysis

---

## Robot Treasure Hunting

Vision-based robot navigation and target searching experiments.

Features:

* Path planning
* Visual target detection
* Obstacle navigation
